# Chat App (client)

This is basic chat.

## Technologies

- React.js
- Socket.io-client

## Run app

1. Clone repository:
``` bash
git clone https://github.com/Taras-Rm/chat-app-practice.git
```

2. Move to client folder:
``` bash
cd ./client
```

3. Install dependencies
``` bash
npm install
```

4. Start app
``` bash
npm run dev
```
